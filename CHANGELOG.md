# Changelog

All notable changes to VerticalSemaphore are documented here.

This project follows semantic versioning.

## 0.1.0 - 2026-06-13

Initial public release.

- Added `VerticalSemaphore`, a vertical SwiftUI close/minimize/zoom control.
- Added `SemaphoreStyle` presets for window chrome and compact tab chips.
- Added macOS `WindowChromeController` for hiding native traffic lights and revealing them on top-left hover.
- Added tab and window usage examples.
- Added README, launch copy, MIT license, tests, and CI.
